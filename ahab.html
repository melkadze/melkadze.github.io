AHAB

An automatic, hosts-based adblocker.

About

Programmed in C#, this automatic hosts-based adblock interface can update your built-in Windows web address blacklist to the latest MVPS hosts file within a simple, clean interface.

Features

This application keeps a backup of your original hosts file, ready to restore it at any point. It can tell if the modified file has already been applied, in order to intelligently keep this backup through updates.

The program also comes with the ability to delete or wipe your hosts file as an added utility.
